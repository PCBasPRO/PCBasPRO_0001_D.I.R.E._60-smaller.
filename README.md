# PCBasPRO_0001_D.I.R.E._60-smaller.
Yet another D.I.R.E. PCB but 60% smaller.

### **Features:**
- Case 60% smaller (50.5mmX63mmX20mm)
- Same display
- Same push buttom location
- New PCB (smaller size)
- new size bottom (10mm)

 ### **My next steps:**
